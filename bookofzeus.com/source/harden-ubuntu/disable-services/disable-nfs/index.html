This the standard network file sharing for Unix/Linux/BSD style operating systems. Unless you require to share data in this manner, disable this.

The following packages are used for the NFS service:

  • nfs-kernel-server
  • nfs-common
  • portmap
  • rpcbind
  • autofs
sudo apt-get purge nfs-kernel-server nfs-common portmap rpcbind autofs
