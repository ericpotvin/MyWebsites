Hide the computer desktop icon

Check the current status:

gsettings get org.gnome.nautilus.desktop computer-icon-visible

If the icon is visible, it will return true, to simply hide it, this settings needs to be false.

gsettings set org.gnome.nautilus.desktop computer-icon-visible false
