Ubuntu comes with some great applications. These applications works great but in some cases, we don't use them or worse, the applications we like or need are not installed.

to fix this issue, there's a very simple way to create your own "after install" or "clean install" script that will remove unwanted applications and install the one you like or need.

Please note this is an example. Please modify to match your needs. SUDO required when installing or removing packages.

Removing unwanted package:

Removing thunderbird

apt-get purge thunderbird-globalmenu thunderbird-gnome-support thunderbird-locale-en thunderbird-locale-en-us thunderbird

Removing totem

apt-get purge totem-mozilla totem-plugins totem

Removing transmission

apt-get purge transmission-common transmission-gtk

Removing empathy

apt-get purge nautilus-sendto-empathy empathy-common empathy

Removing gwibber

apt-get purge libgwibber-gtk2 libgwibber2 gwibber-service-twitter gwibber-service-identica gwibber-service-facebook gwibber-service gwibber

Removing shotwell

apt-get purge shotwell

Removing UbuntuOne

apt-get purge unity-scope-musicstores rhythmbox-ubuntuone ubuntuone-installer ubuntuone-control-panel ubuntuone-couch ubuntuone-client-gnome
apt-get purge libsyncdaemon-1.0-1 gir1.2-ubuntuoneui-3.0 libubuntuoneui-3.0-1 ubuntuone-client

Removing Games

apt-get purge gnomine gnome-sudoku mahjongg

Installing wanted packages:

Custom Apps

apt-get install vim aptitude pidgin xvnc4viewer filezilla gthumb
